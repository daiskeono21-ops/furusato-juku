A3一枚で山形県35市町村の団体一覧をまとめる方法

337団体を35市町村に配分し、A3一枚に収める場合、以下の方法が効果的です。

推奨方法

1. 表形式（最も実用的）

  • レイアウト：縦2列 × 横18行程度で市町村を配置
  • 各セル内容：市町村名＋団体数を数字で明記
  • メリット：データ参照性が高く、印刷にも適している
  • フォント：9～10pt程度で読みやすく
山形市      32団体  |  朝日町     12団体
寒河江市    18団体  |  大江町      8団体
上山市      14団体  |  大石田町   10団体
...（中略）...

2. 棒グラフ + 表併用

  • 上部：横棒グラフで市町村別の団体数を視覚化（数が多い順）
  • 下部：全市町村の数値表
  • メリット：分布が一目瞭然、視覚的インパクト有

3. ヒートマップ風マトリックス

  • 35市町村を5×7マスで配置
  • セルの色濃淡で団体数を表現（濃い＝多い）
  • メリット：地域的な濃淡が直感的
  • 注意：カラー印刷前提

4. 地図ベース（上級）

  • 山形県の簡略地図上に各市町村を配置
  • 団体数を数字またはアイコンで表示
  • メリット：地域的位置関係が理解しやすい
  • デメリット：制作に手間がかかる

最適な選択肢

表形式（方法1） をお勧めします：

  • ✅ 情報密度が高い
  • ✅ 印刷品質が安定
  • ✅ 参照・検索が容易
  • ✅ 最大32団体対応も見やすい

  • 推奨：A3×8～12枚程度で収まる可能性

フォント：6～7pt（新聞並み）

  • Excel/PDFでインタラクティブに閲覧可能

        function createDistrictPage(pres, title, count, colors, districts) {
            let slide = pres.addSlide();
            slide.background = { color: 'ffffff' };
            
            // ヘッダー
            slide.addShape(pres.ShapeType.rect, {
                x: 0, y: 0, w: '100%', h: 0.6,
                fill: { color: colors.title }
            });

                fontFace: 'Yu Gothic'
            });
            
            slide.addText(count, {
                x: 9, y: 0.08, w: 1.5, h: 0.45,
                fontSize: 20, bold: true, color: 'white',
                align: 'right', fontFace: 'Yu Gothic'
            });
            
            // 本体
            let yPos = 0.9;
            let xPos = 0.5;
            let colCount = 0;
            
            districts.forEach(district => {
                const boxHeight = 0.3 + (Math.min(district.groups.length, 10) * 0.25);
                
                // 市町村名
                slide.addShape(pres.ShapeType.rect, {
                    x: xPos, y: yPos, w: 3, h: 0.35,
                    fill: { color: colors.bg },
                    line: { color: colors.title, width: 1 }
                });
                
                slide.addText(`【${district.city}】(${district.count})`, {
                    x: xPos + 0.1, y: yPos + 0.05, w: 2.8, h: 0.25,
                    fontSize: 11, bold: true, color: colors.title,
                    fontFace: 'Yu Gothic'
                });
                
                // 団体リスト
                let innerY = yPos + 0.38;
                district.groups.slice(0, 10).forEach(group => {
                    slide.addText(`・${group}`, {
                        x: xPos + 0.1, y: innerY, w: 2.8, h: 0.24,
                        fontSize: 9, color: '#333',
                        fontFace: 'Yu Gothic'
                    });
                    innerY += 0.25;
                });
                
                yPos += boxHeight + 0.3;
                
                // 改行処理
                if (yPos > 15 && colCount === 0) {
                    yPos = 0.9;
                    xPos = 5.9;
                    colCount = 1;
                }
            });
        }
